# 📊 Análise Estatística: Comparação de Estratégias de Ensino com Teste Z

## 🚀 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de Estatística Inferencial para avaliar a eficácia de duas estratégias de ensino de matemática no ensino médio.

A análise utiliza técnicas de estatística descritiva, testes de normalidade e o Teste Z para verificar se existe evidência estatística de que uma das metodologias proporciona um desempenho superior aos alunos.

Durante o projeto foram realizadas etapas fundamentais de um estudo estatístico profissional, incluindo exploração dos dados, validação de premissas, formulação de hipóteses, interpretação de resultados e visualização gráfica da distribuição da estatística de teste.

---

## 🎯 Objetivo

Determinar se a Estratégia B apresenta desempenho médio superior à Estratégia A, utilizando métodos estatísticos para apoiar a tomada de decisão baseada em dados.

---

## 📚 Conceitos Aplicados

* Estatística Descritiva
* Média e Variância
